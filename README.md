# Android App-音乐播放器
### 功能

- 界面结合DrawerLayout、Toolbar、RecyclerView、FloatingActionBar等组件构成
- 扫描本地音乐形成歌单（Cusor、ContentResolver）
- 点击按钮滑动到顶部
- 点击音乐后显示播放页面
- 切换播放模式（单曲循环、列表循环、随机播放等）
- 切换上/下一首
- 歌词同步滚动显示
- 在状态栏显示通知，App后台运行时可以在通知栏控制音乐播放与停止（Service、BroadcastReceiver、Binder、Handler、多线程等）
